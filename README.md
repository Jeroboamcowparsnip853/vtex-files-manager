## üîß How to Use vtex-files-manager

1. Open your command line interface (Command Prompt on Windows, Terminal on macOS/Linux).
2. Type `vtex-files-manager` and press Enter. This will display a list of available commands.
3. For more information on specific commands, type `vtex-files-manager help`.

### Common Commands

- **Upload File**: `vtex-files-manager upload <file_path>` - Uploads the specified file.
- **Check Status**: `vtex-files-manager status` - Displays the upload status and any error messages.
- **Help**: `vtex-files-manager help` - Provides a guide to using the tool.
